# 玩具项目：微信小程序 NFC 读取演示

本项目为微信小程序示例，支持安卓手机通过NFC读取NTAG213标签内容，并展示读取到的文本信息。

## 功能简介
- 安卓手机通过NFC读取NTAG213标签（NDEF格式）
- 读取内容后在页面展示

## 目录结构
- miniprogram/  小程序主目录
- miniprogram/pages/nfc/  NFC读取页面
- project.config.json  微信开发者工具配置

## 使用说明
1. 用微信开发者工具导入本项目
2. 仅安卓手机支持NFC读取，iOS暂不支持
3. 手机需开启NFC功能
4. 需在真机调试
